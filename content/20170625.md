-> sudo apt update 更新

-> sudo rm /var/lib/dkpg/lock 解開 /var/lib/dkpg/ 的 lock

-> sudo apt upate 確認是否最新

-> sudo apt install nginx 安裝 nginx

-> cd /var/www/html 進入

->ls -l 長形列印

-> sudo vi index.nginx-debian.html 修改
