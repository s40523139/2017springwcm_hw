如何將影片上傳至 fossil

Date 週二 27 六月 2017 By 40523139 Tags fossil

步驟 : mkdir test

-> cd tset

-> fossil clone https://(這裡加帳號就不用再 push ex : 40523139@)mde2a2.kmol.info/wcmg7/wiki?name=wcmg7 text.foosil

-> mk wd

-> cd wd

-> fossil open cd ./../test.fossil( 把影片丟進去 )

-> fossil add . 加入

-> fossil commit -m ""

-> fossil push (剛剛有加帳號就不用在 push)
